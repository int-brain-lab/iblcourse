# IBL Cosyne 2026 Tutorial

11:30 a.m. –12:30 p.m. -  IBL Open Data Platform and Tools
SPEAKER: Olivier Winter and Michael Schartner, International Brain Laboratory

In this tutorial, we demonstrate how to analyse the IBL brain-wide Neuropixels dataset using a compressed representation of the full data. Each of the >50,000 neurons is described by its task-related responses as a concatenation of trial-averaged activity (PETHs). Participants will learn to sort neuronal responses by similarity and relate the resulting patterns to anatomical structure.
